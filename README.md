# Proyecto 3 - Analítica 📊

Proyecto de análisis de datos y ciencia de datos enfocado en proporcionar insights a través de análisis estadístico, visualización de datos y técnicas de analítica avanzada.

## Descripción General

Este repositorio contiene un conjunto de análisis de datos realizados mediante Jupyter Notebooks, con el objetivo de explorar, procesar y analizar conjuntos de datos para extraer conclusiones valiosas y patrones significativos.

## 🛠️ Tecnologías Utilizadas

- **Jupyter Notebook** (98%) - Entorno principal para análisis interactivo
- **Python** (1.8%) - Lenguaje de programación para ciencia de datos
- **CSS** (0.2%) - Estilos para visualización en notebooks

## 📁 Estructura del Proyecto

```
proyecto3_analitica/
├── README.md                 # Este archivo
├── notebooks/               # Jupyter Notebooks con análisis
├── data/                    # Conjuntos de datos
├── scripts/                 # Scripts Python de utilidad
└── outputs/                 # Resultados y visualizaciones
```

## 🚀 Características

- ✅ Análisis exploratorio de datos (EDA)
- ✅ Visualización interactiva de datos
- ✅ Procesamiento y limpieza de datos
- ✅ Análisis estadístico y modelado
- ✅ Generación de reportes

## 📋 Requisitos

Para ejecutar este proyecto, necesitas tener instalado:

- Python 3.7+
- Jupyter Notebook
- Librerías de análisis de datos:
  - pandas
  - numpy
  - matplotlib
  - seaborn
  - scipy

## 📦 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/valentinabc19/proyecto3_analitica.git
cd proyecto3_analitica
```

2. Instala las dependencias:
```bash
pip install -r requirements.txt
```

3. Inicia Jupyter Notebook:
```bash
jupyter notebook
```

## 💡 Uso

Abre cualquiera de los Jupyter Notebooks en el directorio del proyecto para ver los análisis interactivos. Cada notebook contiene:
- Carga y exploración de datos
- Visualizaciones
- Análisis estadísticos
- Conclusiones y hallazgos

## 📊 Análisis Disponibles

Los análisis incluyen pero no se limitan a:
- Estadística descriptiva
- Correlaciones y relaciones entre variables
- Tendencias temporales
- Segmentación y clustering
- Predicciones y modelado
